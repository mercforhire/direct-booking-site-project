# Plan 18-02: Admin Scoping

## Goal
Scope all admin server actions and admin RSC pages to the logged-in admin's landlord, so each admin only sees and manages their own rooms, bookings, and settings.

## must_haves
- [ ] All admin server actions use `getLandlordForAdmin()` instead of bare `requireAuth()`
- [ ] `createRoom` assigns `landlordId` from the admin's landlord
- [ ] `updateRoom`, `deleteRoom` verify the room belongs to the admin's landlord before modifying
- [ ] All booking-admin actions verify `booking.room.landlordId` matches the admin's landlord
- [ ] `upsertSettings` uses `landlordId` instead of `id: "global"`
- [ ] All 7 admin RSC pages filter queries by `landlordId`
- [ ] Admin sidebar displays the landlord name
