# Plan 17-03: Sign-Up CTAs on Login Page + Home Page

## Goal
Guests can discover account sign-up from two additional surfaces: the `/guest/login` page (link below the form) and the home page footer strip (secondary CTA alongside "My Bookings →").

## Scope
- `src/app/guest/login/page.tsx` — add "Don't have an account?" link
- `src/app/page.tsx` — add "Create account" link in footer strip

---

## Tasks

### Task 1 — Login page: add sign-up link

**File:** `src/app/guest/login/page.tsx`

After the "Forgot password?" `<div>` block (lines ~299–315), add a sign-up nudge using the same `forgot-link` class and style:

```tsx
{/* Sign-up nudge */}
<div style={{ textAlign: "center", marginTop: "0.5rem" }}>
  <Link
    href="/guest/signup"
    className="forgot-link"
    style={{
      fontSize: "0.68rem",
      letterSpacing: "0.12em",
      textTransform: "uppercase",
      opacity: 0.3,
      color: "#f0ebe0",
      textDecoration: "none",
      transition: "opacity 0.2s ease",
    }}
  >
    No account? Create one
  </Link>
</div>
```

Place this inside the `<form>` block, right after the "Forgot password?" div.

---

### Task 2 — Home page: add "Create account" link in footer strip

**File:** `src/app/page.tsx`

Locate the "returning guest strip" section (around lines 501–541). It currently has:
- A `<div>` with "Already booked with us?" text
- A single `<Link>` "My Bookings →" ghost button

Add a "Create account" text link after the existing ghost button. Update the right-side section to show both:

```tsx
{/* Right side — two CTAs */}
<div style={{ display: "flex", alignItems: "center", gap: "1rem", flexWrap: "wrap" }}>
  <Link
    href="/guest/signup"
    className="ghost-btn"

      fontSize: "0.68rem",
      letterSpacing: "0.12em",
      textTransform: "uppercase",
      opacity: 0.4,
      color: "#f0ebe0",
      textDecoration: "none",
      transition: "opacity 0.2s ease",
    }}
  >
    Create account
  </Link>
  <Link
    href="/guest/login?next=/my-bookings"
    className="ghost-btn"
    style={{
      border: "1px solid rgba(255,255,255,0.2)",
      color: "rgba(240,235,224,0.65)",
      padding: "0.55rem 1.4rem",
      borderRadius: "9999px",
      fontSize: "0.72rem",
      letterSpacing: "0.14em",
      textTransform: "uppercase",
      textDecoration: "none",
      transition: "background 0.2s ease",
      whiteSpace: "nowrap",
    }}
  >
    My Bookings →
  </Link>
</div>
```

The left-side "Already booked with us?" text div stays unchanged.
