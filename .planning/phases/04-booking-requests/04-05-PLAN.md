**`src/app/bookings/[id]/page.tsx`** — RSC, dual access (Pattern 5 from research doc):

```typescript
export const dynamic = "force-dynamic"
```

Logic:
1. Await `params.id` and `searchParams.token` + `searchParams.new`
2. `createClient()` + `supabase.auth.getUser()` to check session
3. `prisma.booking.findUnique({ where: { id }, include: { room: { select: { name: true, location: true, addOns: { select: { id: true, name: true, price: true } } } } } })`
4. `notFound()` if no booking
5. Auth gate:
   - `hasAuth = user && booking.guestUserId === user.id`
   - `hasToken = token && token === booking.accessToken`
   - If neither: `redirect("/guest/login?next=/bookings/" + id)`
6. Coerce `estimatedTotal = Number(booking.estimatedTotal)` and add-on prices at RSC boundary
7. Render `<BookingStatusView booking={...} showSuccessBanner={searchParams.new === "1"} />`
