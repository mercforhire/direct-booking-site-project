    **createStripeCheckoutSession(bookingId: string)**:
    - Call requireAuth() (same pattern as booking-admin.ts — admin not required, this is guest action; actually NO requireAuth needed for guest action — just validate booking exists)
    - NOTE: createStripeCheckoutSession is called by the guest (not admin), so no requireAuth(). Validate via DB lookup only.
    - Fetch booking: prisma.booking.findUnique({ where: { id: bookingId, status: "APPROVED" }, include: { room: { select: { name: true } } } })

      - unit_amount: Math.round(Number(booking.confirmedPrice) * 100)  — MUST be integer cents

    - success_url: `${origin}/bookings/${booking.id}?paid=1`
    - cancel_url: `${origin}/bookings/${booking.id}`
    - customer_email: booking.guestEmail
    - Update booking: prisma.booking.update({ where: { id: bookingId }, data: { stripeSessionId: session.id } })
