# Plan 17-01: Guest Signup Server Action + `/guest/signup` Page

## Goal
Guests can create an account at `/guest/signup` (name, email, phone, password), are auto-confirmed, and are redirected to `/my-bookings` after sign-in. The page visually mirrors `/guest/login`.

## Scope
- New server action: `src/actions/auth.ts` → `createGuestAccount`
- New page: `src/app/guest/signup/page.tsx`

---

## Tasks

### Task 1 — `createGuestAccount` server action

**File:** `src/actions/auth.ts` (new file)

Create a `"use server"` module with a single export:

```ts
export async function createGuestAccount(data: {
  name: string
  email: string
  phone: string
  password: string
}): Promise<{ error: string } | { success: true }>
```

**Logic:**
1. Validate inputs (name non-empty, valid email, phone non-empty, password ≥ 8 chars) — return `{ error }` on failure
2. Create admin client exactly as in `src/actions/booking.ts`:
   ```ts
   const adminClient = createSupabaseClient(
     process.env.NEXT_PUBLIC_SUPABASE_URL!,
     process.env.SUPABASE_SERVICE_ROLE_KEY!
   )
   ```
3. Call:
   ```ts
   const { error: createError } = await adminClient.auth.admin.createUser({
     email: data.email,
     password: data.password,
     email_confirm: true,
     user_metadata: { role: "guest", name: data.name, phone: data.phone },
   })
   ```
4. If `createError`:
   - If the error message contains "already registered" / "already exists" / "duplicate" → return `{ error: "An account with this email already exists." }`
   - Otherwise → return `{ error: "Could not create account. Please try again." }`
5. Return `{ success: true }`

**Import:** `import { createClient as createSupabaseClient } from "@supabase/supabase-js"`

---

### Task 2 — `/guest/signup` page

**File:** `src/app/guest/signup/page.tsx` (new file)

Mirror `/guest/login/page.tsx` exactly — same outer shell, same CSS, same `Suspense` + inner function component split. Differences:

**Headline:**
```jsx
Create <span style={{ color: "#d4956a" }}>Account</span>
```

**Fields** (instead of email + password):
1. Name — `type="text"`, `autoComplete="name"`, label "Name"
2. Email — `type="email"`, `autoComplete="email"`, label "Email"
3. Phone — `type="tel"`, `autoComplete="tel"`, label "Phone"
4. Password — `type="password"`, `autoComplete="new-password"`, label "Password"

**State:** `name`, `email`, `phone`, `password`, `error`, `loading`

**Submit handler:**
```ts
async function handleSubmit(e: React.FormEvent) {
  e.preventDefault()
  setError(null)
  setLoading(true)

  const result = await createGuestAccount({ name, email, phone, password })

  if ("error" in result) {
    setError(result.error)
    setLoading(false)
    return
  }

  // Auto sign-in after account creation
  const supabase = createClient()  // browser client
  const { error: signInError } = await supabase.auth.signInWithPassword({ email, password })
  setLoading(false)
  if (signInError) {
    setError("Account created but sign-in failed. Please sign in manually.")
    return
  }

  window.location.href = "/my-bookings"
}
```

**Below the submit button** — "Already have an account?" link:
```jsx
<div style={{ textAlign: "center", marginTop: "-0.75rem" }}>
  <Link
    href="/guest/login"
    className="forgot-link"
    style={{ /* same as forgot-link style */ }}
  >
    Already have an account? Sign in
  </Link>
</div>
```

**Back link** at bottom (same as login): `← Back to rooms` → `/`

**Button label:** "Create Account" (loading: "Creating…")

**No** `justReset` banner or `useSearchParams` needed — keep `Suspense` wrapper for consistency.
